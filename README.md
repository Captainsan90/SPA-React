# React + Vite

Setup Instructions

Install Dependencies: Open your terminal in the project root and run: npm install, npm run dev

Usage
Adding Projects: Enter a title and description in the "Add Project" form and click the Add button. The new project will appear in the list below.

Searching: Use the "Search Projects" input field to filter your projects by title in real-time.


Features
Modular Component Architecture: Built using reusable components (ProjectForm, ProjectList, ProjectItem).

Real-time Filtering: Dynamic search functionality that updates the UI as you type.

State Management: Utilizes React useState hooks to handle data flow and user input.

Mockup-Accurate Design: Mimics the provided wireframe with a clean, border-focused aesthetic.

⚠️ Known Limitations
Non-Persistent Storage: Data is stored in local component state. Refreshing the browser will reset the project list to the default items.
